# 财富与契约插件 v1.0
一款专为AstrBot设计的群聊插件，融合了经济系统、契约系统（性奴买卖）、道具系统以及社交系统，为群聊提供丰富的互动玩法。
## ✨ 核心功能简介
### 📊 财富系统
- 每日签到获取金币，连续签到有额外奖励
- 银行存款每日1%利息
- 金币排行榜展示总资产前10名
- 资产核查功能，详细展示用户资产
### ⛓️ 契约系统
- 购买其他用户作为性奴
- 出售拥有的性奴
- 性奴排行榜展示拥有性奴数量前10名
- 赎身功能，支付1.5倍身价解除契约
- 永久绑定性奴（使用驯服贴道具）
### 🧰 道具系统
- 签到商店购买各种道具
- 道具背包查看已拥有的道具
- 道具包括：
  - 驯服贴：永久绑定性奴
  - 强制购买符：强制购买已有主人的性奴
  - 自由身保险：24小时内不被购买为性奴
  - 红星制裁：对全群高资产/多性奴用户进行制裁
  - 市场侵袭：对指定用户发起侵袭掠夺
  - 缔结关系道具（卡天亚戒指、一壶烈酒、黑金卡）
  - 礼物（玫瑰花束、定制蛋糕）
### 💕 社交系统
- 约会系统：发起约会邀请，触发随机事件影响好感度
- 好感度系统：根据好感度划分关系等级
- 特殊关系：缔结恋人、兄弟、包养关系
- 赠送礼物增加好感度
## 🧾 指令列表
### 签到与经济
- `/签到`：每日签到获取金币
- `/签到查询`：查看当前签到状态和预计收益
- `/存款 <金额>`：将现金存入银行
- `/取款 <金额>`：从银行取出金币
- `/转账 <金额> @对方`：转账给其他用户（收取10%手续费）
- `/签到买彩票`：花费5金币购买彩票（2%中奖概率）
- `/金币排行榜`：显示本群金币总资产前10名
- `/性奴排行榜`：显示本群拥有性奴数量前10名
- `/资产核查@用户`：查询用户的资产详情
### 契约系统
- `购买@用户`：购买其他用户作为性奴
- `出售@用户`：出售你拥有的性奴
- `/我的契约`：查看当前契约关系（主人和性奴）
- `/看看你的契约@用户`：查看指定用户的契约关系
- `/看看详细契约@用户`：查看用户的详细契约信息
- `/赎身`：支付1.5倍身价解除契约关系
- `/打劫@用户`：打劫其他用户的金币（60分钟冷却）
### 打工系统
- `/打工 工作名 @用户`：让性奴打工赚钱
- `/一键打工 工作名`：让所有性奴同时进行指定工作
- `/打工列表`：显示可用的工作列表
### 道具系统
- `/签到商店`：查看可购买的道具列表
- `/签到背包`：查看自己拥有的道具
- `/签到商店购买 <道具名> [数量]`：购买道具
- `/道具驯服贴 @用户`：永久绑定性奴
- `/道具强制购买 @用户`：强制购买已有主人的性奴
- `/道具自由身保险`：激活24小时自由身保护
- `/道具红星制裁`：对全群高资产/多性奴用户进行制裁（每天限用1次）
- `/道具市场侵袭 @用户`：对指定用户发起侵袭掠夺（每小时限用1次）
### 社交系统
- `/约会@对方`：向其他用户发起约会邀请
- `/同意约会`：同意对方的约会邀请
- `/缔结关系@对方 关系类型`：与对方缔结特殊关系（恋人、兄弟、包养）
- `/解除关系@对方`：解除与对方的特殊关系
- `/查看关系@对方`：查看你与对方的关系状态
- `/社交网络`：查看你的社交关系网络（按好感度排序）
- `/赠送礼物@对方 礼物名`：赠送礼物增加好感度
### 帮助
- `/签到帮助`：显示完整帮助菜单
## 📁 项目结构
```
Astrbot/
    │-- data/                    # 数据存储目录
        │-- plugins_WealthAndContract_data/
            │-- WAC_data.yml        # 主数据文件
            │-- WAC_propdata.yml    # 道具数据文件
            │-- WAC_social_data.yml # 社交数据文件
            │-- WAC_time_data.yml   # 时间数据文件
            │-- WAC_operations.log  # 操作日志
        |-- plugins/
            │-- astrbot_plugin_WealthAndContract/
                │-- main.py                  # 插件主程序
                │-- metadata.yaml            # 插件元数据
                │-- requirements.txt         # 依赖列表
                │-- 请以你的名字呼唤我.ttf    # 字体文件
                │-- images/                  # 图片缓存目录
```
## 📜 使用说明
1. 将插件文件夹放入AstrBot的`data/plugins`目录
2. 安装依赖：`pip install -r requirements.txt`
3. 重启AstrBot，在群聊中使用指令即可
## 📌 注意事项
- 插件更新时可能连带数据文件格式一同更新，如果更新完出现报错请删除`data/plugins_WealthAndContract_data/`数据目录下的所有数据文件
- 插件需要网络连接以获取头像和背景图片
- 确保`data/plugins_WealthAndContract_data`目录有写入权限
- 如果缺少字体文件，请手动添加
- 本插件与[zgojin/astrbot_plugin_Qsign](https://github.com/zgojin/astrbot_plugin_Qsign)会产生冲突，请不要同时安装
